<h3>🚀 Key Features</h3>
<ul>
  <li><strong>Hybrid Extraction Grid:</strong> Pulls media feeds cleanly via target-specific links or direct injection configurations.</li>
  <li><strong>Silent System Architecture:</strong> Runs smoothly out of the Windows system tray without crowding the taskbar.</li>
  <li><strong>Adaptive Timing Engine:</strong> Integrated rate-limiting delays to preserve optimal connection performance.</li>
  <li><strong>Single Instance Lock:</strong> Intelligent local port management prevents background process conflicts.</li>

  <h3 style="margin-top: 0; color: #58a6ff;">🛠️ How to Build from Source Code</h3>
  <p>Follow these steps to compile the raw application files into a distribution executable on your local machine:</p>

  <ol>
    <li style="margin-bottom: 15px;">
      <strong>Check System Prerequisites</strong>
      <p style="margin: 5px 0 0 0; color: #8b949e;">Ensure your computer has <a href="[https://nodejs.org/](https://nodejs.org/)" style="color: #58a6ff;">Node.js</a> installed to manage background packages.</p>
    </li>
    
    <li style="margin-bottom: 15px;">
      <strong>Clone the Local Repository</strong>
      <p style="margin: 5px 0 5px 0; color: #8b949e;">Launch your command terminal and pull down the files:</p>
      <pre style="background-color: #0d1117; padding: 10px; border-radius: 6px; border: 1px solid #21262d; color: #c9d1d9;"><code>git clone [https://github.com/goldydrop/Pin-Sniper.git](https://github.com/goldydrop/Pin-Sniper.git)
cd Pin-Sniper</code></pre>
    </li>

    <li style="margin-bottom: 15px;">
      <strong>Initialize Dependencies</strong>
      <p style="margin: 5px 0 5px 0; color: #8b949e;">Download and synchronize the required modules:</p>
      <pre style="background-color: #0d1117; padding: 10px; border-radius: 6px; border: 1px solid #21262d; color: #c9d1d9;"><code>npm install</code></pre>
    </li>

    <li style="margin-bottom: 5px;">
      <strong>Compile the Distribution Package</strong>
      <p style="margin: 5px 0 5px 0; color: #8b949e;">Generate the production Windows desktop installer application:</p>
      <pre style="background-color: #0d1117; padding: 10px; border-radius: 6px; border: 1px solid #21262d; color: #c9d1d9;"><code>npm run dist</code></pre>
      <p style="margin: 5px 0 0 0; font-size: 13px; color: #34d058;">✨ Output file will be generated instantly inside your local <code>/dist</code> directory!</p>
    </li>
  </ol>
</div>
